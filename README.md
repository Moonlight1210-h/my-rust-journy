## 🚀 لماذا راست؟ (The Why)
بعد سنوات في "جنة" بايثون حيث السهولة والمرونة، قررت دخول عالم راست بحثاً عن:
- **أمان الذاكرة (Memory Safety):** وداعاً للـ Null pointer والـ Segmentation faults.
- **الأداء العالي:** سرعة تقترب من C++ مع أمان أعلى.
- **التوثيق المذهل:** نظام `cargo doc` الذي يبني لك موقعاً كاملاً لكودك.

## 📝 ما الذي ستجده هنا؟
لقد قمت بتوثيق المفاهيم الأساسية التي تعلمتها، مع أمثلة عملية على:
- [x] **Formatting & Printing:** تنسيق النصوص المتقدم (Padding, Alignment, Base Conversion).
- [x] **Project Structure:** كيف تنظم مشروعك باستخدام الـ Modules والملفات المنفصلة.
- [x] **Argument Checking:** كيف يحميك المترجم (Compiler) من أخطاء وقت التشغيل.
